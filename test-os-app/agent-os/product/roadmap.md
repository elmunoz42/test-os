# Product Roadmap

## Phase 1: MVP

- Application shell with dark sidebar navigation.
- Test Foundation wizard for charter, coverage map, oracle data, test standards, and Hermes config.
- Section Validation authoring workspace.
- Verdict & Reporting dashboard.
- Deployment & Automation setup and status dashboard.

## Phase 2: Post-Launch

- Replace browser `localStorage` persistence with a database-backed workspace API.
- Persist generated artifacts and oracle fixtures outside the app-under-test repository.
- Add authenticated multi-workspace support.
- Add live Hermes execution and EC2 status integrations.
